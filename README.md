## できること

- Web制作ディレクション（要件整理／進行管理／品質管理）
- アクセス解析を前提とした設計・改善提案

仕様が固まりきっていない段階や、  
途中で整理・立て直しが必要なフェーズからの参画も可能です。

- 上級ウェブ解析士として、感覚ではなくデータを基に判断
